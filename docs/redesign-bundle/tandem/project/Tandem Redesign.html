Tandem · interface redesign · 2026

A document in conversation
with the writer beside you.

The current Tandem chrome is workmanlike but loud — buttons live far from where the writing happens, and Claude's presence reads as a side panel rather than a co-author. This redesign brings tools to the cursor, settles the chrome down, and lets the prose breathe. Every artboard below is an actual desktop window state.

01
Selection mini-toolbar. Highlight any text and the formatting + Highlight / Comment / Ask Claude / Flag tools surface in place. The cursor never has to leave the page.
02
Authorship gutter. A 2px thread to the left of every paragraph shows who last touched it — blue for you, coral for Claude. Pulses softly when Claude is reading.
03
Quieter chrome. A single thin top bar with brand, doc title, mode toggle, and panel controls. Tabs are a 30px secondary row. No shadows except things that float.
04
Editorial typography. Source Serif 4 for prose, Inter Tight for chrome, JetBrains Mono for status and timestamps. All overrideable in Tweaks.
