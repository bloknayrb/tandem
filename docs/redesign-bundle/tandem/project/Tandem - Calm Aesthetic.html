Tandem · calm aesthetic exploration · 2026-05-12

Less line, more air.
Five directions toward a calmer surface.

v3 today separates regions almost entirely with 1px hairlines: titlebar, tabs, fmt bar, rail edge, every annotation card, every banner. The hairlines are doing all the work because the surface tones are nearly identical. The result reads as busy.

These five proposals each remove that load differently — some keep the docked rail and strip its borders, some detach panels to float, one moves notes into the margin and dispenses with the rail entirely. They share the warm paper canvas, the serif editor, the authorship colors, and the type scale — only the chrome treatment differs, so you can mix freely.

What's busy today

  • Five horizontal hairlines stacked top-to-bottom: titlebar → tabs → fmt bar → main → status bar
  • Rail joins on left edge with another 1px, then again at every tab header
  • Every annotation card has a 1px border, gaining colored accent borders on hover/active
  • Banners (held, info, conn) repeat: tinted fill + tinted 1px border + radius
  • Chrome (--surface-muted, 0.975) is almost the same tone as bg (0.985) — borders carry all the separation

Calming moves shared by all

  • Warmer paper canvas (oklch 0.965 → 0.94) so a clean sheet at 0.995 actually reads as "lifted"
  • Tone difference replaces hairlines wherever possible — chrome is the canvas, editor is the sheet
  • Author color, kind tint, and weight carry meaning previously carried by borders
  • Shadows do separation only when something is actually floating (sheet, popover, frost)
  • Status bar becomes mono labels on the canvas — no top border
