8 — Design Token Alignment

Repo: CSS custom properties on :root
Token naming mismatch
Design: --accent, --ink, --surface, --hair
Repo: --tandem-accent, --tandem-fg, --tandem-surface…
Both sides
Design says
  • Design uses shorthand tokens: --accent, --ink, --ink-faint, --surface, --hair, --font-serif, --font-sans, --font-mono
Repo today
  • Repo uses namespaced tokens: --tandem-accent, --tandem-fg, --tandem-fg-muted, --tandem-fg-subtle, --tandem-surface, --tandem-border, etc.
  • These are more complete and production-safe; design should adopt the --tandem-* namespace rather than the other way around
  • Missing in repo: --tandem-editor-font-family, --tandem-density
